In case someone asks "Why Bootstrap?"


Mobile-first approach
  • In Bootstrap, mobile-first styles are part of the core framework
  • Because it is convenient for users that want to access your site or web app via phone
Browser compatibility
  • Bootstrap 5 is compatible with all modern browsers (Chrome, Firefox, Edge, Safari, and Opera).
  • Yes, no more "We only support Chrome"
Responsive features
  • Bootstrap's responsive CSS adjusts to phones, tablets, and desktops
  • Resize this browser window to see the effect like on a phone screen

Use the contextual background classes to provide "meaning through colors".
